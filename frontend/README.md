# Emergency Response Intelligent System (ERIS) - Frontend

This is the front-end application for the Emergency Response Intelligent System, built with [React](https://reactjs.org/) and powered by [Vite](https://vitejs.dev/).

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

Navigate to the `frontend` directory and install the dependencies:

```bash
npm install
```

### Development Server

To start the development server with Hot Module Replacement (HMR), run:

```bash
npm run dev
```
or 
```bash
npm start
```

The application will be available at `http://localhost:3000`.

### Production Build

To bundle the application into static files for production, run:

```bash
npm run build
```

The optimized files will be generated in the `dist` directory.

### Testing

To run the unit tests via Vitest:

```bash
npm test
```
